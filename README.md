# Programacion_Tradicional_vs_POO_Clima

Programación Tradicional: Se basa en funciones que procesan datos de entrada de forma secuencial. Es más simple para tareas pequeñas pero más difícil de mantener si el sistema crece.

POO: Organiza el código en objetos que agrupan datos (temperaturas) y comportamientos (cálculo de promedio). Facilita el encapsulamiento y la organización lógica del código.
